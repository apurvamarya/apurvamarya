### 🎯 FaceTrack — AI Attendance System

### 🌿 EcoAir — Air Quality Dashboard
[![GitHub](https://img.shields.io/badge/GitHub-Repository-181717?style=flat-square&logo=github&logoColor=white)](https://github.com/apurvamarya/ecoair-dashboard)
